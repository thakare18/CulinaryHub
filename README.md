## Project Overview

CulinaryHub is a frontend web application that delivers an interactive and engaging user experience for culinary content. The platform showcases recipes, food categories, and related features through a modern and responsive interface. The project is built using contemporary web technologies and follows best practices for performance and maintainability.

## Live Demo

The application is deployed and accessible at:
https://culinaryhub.netlify.app/

## Tech Stack

Frontend: React.js with Vite

Languages: JavaScript, HTML, CSS

Styling: CSS modules(Tailwind) / utility-first approach

Deployment: Netlify

## Features

Responsive and visually appealing UI

Dynamic presentation of culinary content

Fast performance using optimized frontend tooling

Deployment with continuous delivery via Netlify

## Installation & Setup
Prerequisites

Node.js (v16+ recommended)

npm or yarn

Clone and Setup
git clone https://github.com/thakare18/CulinaryHub.git
cd CulinaryHub
npm install
npm run dev


Once started, the application will be available at:

http://localhost:5173/

Folder Structure
CulinaryHub/
│
├── public/                 
├── src/                   
│   ├── components/
│   ├── pages/
│   ├── styles/
│   └── main.jsx
├── .gitignore
├── index.html
├── package.json
└── vite.config.js
